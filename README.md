During this challenge, I reinforced my knowledge of CSS Grid, especially how to create complex layouts that change based on screen size. I also practiced using pseudo-elements for the colorful borders on top of the cards.

I learned how to use `grid-column` properly by ensuring the parent has `display: grid`.
